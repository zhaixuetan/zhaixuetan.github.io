try-except : 处理异常

1. try-except捕获异常：

Something went wrong!

2. try-except 捕获具体异常：

Something went wrong! [Errno 2] No such file or directory: 'test.py'

3. try-except 对异常进行分别处理： 

There is a FileNotFoundError! [Errno 2] No such file or directory: 'test.py'
